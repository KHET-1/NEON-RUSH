<h3 align="center">NEON RUSH</h3>
<p align="center">A three-perspective arcade racer with boss rush mechanics — 100% procedural, zero external assets.</p>

<p align="center">
  <b>Desert Velocity</b> (vertical) → <b>Excitebike</b> (side) → <b>Micro Machines</b> (top-down) → <b>Victory</b>
</p>

---

## What It Is

~7,700 lines of pure Python + Pygame. Every visual, sound, and music track is synthesized at runtime. Three completely different perspectives, each with its own physics and boss. Beat them all, then do it again at higher tiers.

No sprites. No audio files. Just math, code, and velocity.

## Status

Playable. Procedural generation and multi-perspective design are the core strengths. A joyful, self-contained artifact from the forge.

## Philosophy

Ship beautiful constraints. The same spirit that drives the Creation Game and the Fire Axe: make something that feels alive with the smallest possible surface.

Part of the larger work being built for Firestar.
